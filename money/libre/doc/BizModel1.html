Бизнес модели проекта LibreMoney

  • Для инвесторов в технологические IT стартапы
  • Для предпринимателей (стартаперов, авторов крауд-проектов)
  • Для инвесторов в Lm-деньги
  • Для существующих краудфандинговых и краудинвестинговых площадок
  • Для обменников, магазинов и др online-сервисов, принимающих платежи
  • Для людей, желающих сберегать накопления в ликвидной валюте без инфляции и без скачков


Для инвесторов в технологические IT стартапы

Партнеры
- Инвесторы (соинвесторы, частные бизнес ангелы, фонды)
- Предприниматели (стартаперы)
- Площадки краудфандинга и краудинвестинга 		Деятельность
- Создание системы (исходный код)
- Создание продукта (Lm-деньги)
- Распространение, популяризация, маркетинг
Ресурсы
- Люди
- Lm-деньги
- Бренд (libremoney)
- Учетные записи в соц.сетях, на форумах и пр. 		Продукт
- Lm-деньги
- Помощь выбора перспективных проектов для инвестирования
- Транзакции денежных средств
- Консультирование 		Взаимоотношенния с клиентом
- Личные консультации
- Сообщества
- Помощь в поиске перспективного проекта для инвестирования
Каналы
- SMM
- Форумы предпренимателей
- Форумы банковских и финансовых институтов (forumprobanki.ru, credity.name)
- Инвесторы в IT стартапы (offline)
- Стартап-мероприятия (offline)
- Партнерские каналы распространения (?) 		Клиенты
- Бизнес-ангелы инвестирующие в IT стартапы на этапе идеи, pre-seed, seed (бизнес-ангелы, венчурные фонды занимающиеся микро-финансированием)
- Венчурные фонды занимающиеся микро-финансированием в IT стартапы на этапе идеи, pre-seed, seed с целью поиска проектов для полноценного инвестирования на последующем этапе
- Люди, интересующиеся OpenSource и готовые финансировать интересные технологические проекты (модель P2P, краудинвестинг)
- Специалисты-профессионалы, желающие инвестировать своё время, помогая в реализации интересных проектов за долю в компании (краудсорсинг)
(Во всех случаях автор проекта отдает краудинвесторам не более 10% проекта. Выход при краудинвестинге возникает в момент выкупа доли крупным фондом. При этом фонд выкупает 10% у краудинвесторов целиком, за одну транзакцию.)
Расходы
- Оплата разработчиков (программистов)
- Оплата маркетологов (изготовление видео, написание статей, SMM и пр)
- Хостинг 		Доходы
- Продажа Lm-денег из первоначальной эмисии (генезиса)
- % от собранных средств на площадке libremoney.com

Для существующих краудфандинговых и краудинвестинговых площадок

Партнеры
- Инвесторы (соинвесторы, частные бизнес ангелы, фонды)
- Предприниматели (стартаперы)
(Существующие площадки являются прямыми конкурентами для libremoney.com. Предполагается, что развитие libremoney.com не получит, а будет служить резервным вариантом для управления своим кошельком и проектами. Предполагается, что люди будут пользоваться системой LibreMoney не подозревая об этом. С прозрачным обменом фиатных денег на Lm-деньги и обратно на крауд-площадках.) 		Деятельность
- Создание системы (исходный код)
- Создание продукта (Lm-деньги)
- Распространение, популяризация, маркетинг
Ресурсы
- Люди
- Lm-деньги
- Бренд (libremoney)
- Учетные записи в соц.сетях, на форумах и пр. 		Продукт
- Lm-деньги
- API подключения к системе для краудфандинговых площадок
- Транзакции денежных средств
- Консультирование
(Самое важное для существующих площадок: возможность привлечения средств из любой точки мира, минуя каких либо регуляторов. При этом площадка продолжает зарабатывать, взимая процент от собранных средств. И, дополнительно, получая процент от эмиссии Lm-денег, в случае успешного завершения крауд-проекта.) 		Взаимоотношенния с клиентом
- Личные консультации
- Сообщества

- Краудфандинговые площадки (Kickstarter, Indiegogo, etc)
